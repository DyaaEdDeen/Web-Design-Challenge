Welcome, Challenger!

This is your first real-world project simulation. Your mission is to replicate a 3-page website using only HTML, CSS, and JavaScript. No frameworks, no libraries, just pure code.

The Milestones & Rewards

Milestone 1: The Landing Page (Prize: 200 EGP)
Replicate the home page. This includes the navigation bar, a full-width hero banner, and a simple content section. The focus here is on solid HTML structure and basic CSS styling.
Milestone 2: The About Page (Prize: 500 EGP)
Replicate the "About Us" page. This level introduces more complex CSS layouts using Flexbox to create a responsive card grid. You'll also need to handle hover states.
Milestone 3: The Interactive Sign-Up Form (Prize: 1000 EGP)
Replicate the "Sign Up" page and implement its client-side validation logic using JavaScript. The form must check for valid inputs (email format, password strength) and display user-friendly error messages without reloading the page.

Rules of Engagement

  • Deadline: You have one week to complete as much as you can.
  • Allowed Tools: You can use any search engine (Google, etc.), and read any documentation (MDN is your best friend!).
  • AI Policy: No heavy use of AI for code generation. Your code will be reviewed. Using it for simple debugging or explanation is acceptable.
  • Submission: You will submit your project folder for review.
